Real-Time Image Captioning and Speech Generation

Built an AI tool that converts visual scenes into audio descriptions using image captioning and text-to-speech, enhancing accessibility for visually impaired.

Motivation: People with vision impairments often struggle to access visual information, as screen readers can't interpret images or scenes. This project bridges that gap by using AI to generate captions.

Key Contributions

  • Developed an image captioning system combining ResNet50/VGG16 for feature extraction with LSTM networks for generating descriptive, human-like captions.
  • Integrated text-to-speech (TTS) conversion to vocalize captions in multiple languages, enabling real-time auditory feedback.
  • Used Flickr8k and Flickr30k datasets, preprocessed with custom tokenization, lowercasing, and word-to-index mapping, then mapped image features to captions using pre-trained GloVe 300D embeddings.
  • Evaluated model performance using BLEU scores and improved results through fine-tuning and larger dataset training (30k images).
  • Built an interactive Streamlit web interface to capture, caption, and read out images in real-time for end-user accessibility.

Tools & Technologies

Python, TensorFlow, Keras, OpenCV, Streamlit, ResNet50, VGG16, LSTM, Flickr8k & Flickr30k, BLEU Score, PyTorch, Text-to-Speech APIs

Impact

Empowered visually impaired users to experience the visual world through spoken image descriptions, improving accessibility, independence, and interaction with their environment in real-time.
