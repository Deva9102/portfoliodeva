    Experience

    July 2023

    Deep Learning Intern – National University of Singapore (NUS)

    As a Deep Learning Intern at National University of Singapore, I developed VizCap, a real-time image captioning and text-to-speech system to assist visually impaired users. Trained deep learning models using ResNet50 + LSTM and deployed the solution using AWS SageMaker, Lambda, and Polly, ensuring seamless cloud-based processing and multilingual audio output. Integrated the model into a Streamlit web application and optimized performance under real-world deployment constraints.

    Jan 2023 - March 2023

    AI Intern – VLOG Innovations

    As an AI intern, I was responsible for building an automated visual inspection system for printed circuit boards. I developed a YOLOv5-based object detection model to identify six types of defects and built a Tkinter GUI for visualization and severity analysis. The model delivered reliable performance across over 100 annotated PCB images, validating its effectiveness in defect detection.
