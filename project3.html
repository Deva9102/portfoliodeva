AI-Powered Oral Tissue Compatibility Evaluation for Dental Implants


Designed a deep learning–based diagnostic system that analyzes ultrasonography images to evaluate oral tissue compatibility for dental implants.


Motivation: Early implant failures caused by soft tissue issues are more common than late failures, driving this research to improve tissue compatibility assessment using AI and ultrasonography.

Key Contributions

  • Built a hybrid model combining Convolutional Neural Networks (CNNs) and Deep Belief Networks (DBNs) to extract and refine complex tissue patterns from ultrasonography images.
  • Implemented classification models using SVM, Random Forest, and Decision Tree, achieving 87% accuracy individually, and enhanced performance to 90.3% using Gradient Boosting ensemble learning.
  • Applied Grad-CAM visualizations to highlight the key regions in ultrasound scans influencing predictions.
  • Developed a real-time web interface allowing doctors to upload ultrasound images and receive instant compatibility predictions.
  • Curated a high quality dataset covering various oral pathologies (e.g., parotitis, sialadenitis, Sjögren’s disease) with preprocessing techniques such as histogram equalization, pixel normalization, and noise reduction.

Tools & Technologies

Python, TensorFlowm, PyTorch, Scikit-learn, OpenCV, Grad-CAM, Pandas, Matplotlib, Web UI (Streamlit)
