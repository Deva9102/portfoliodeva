PCB Defect Detection using YOLOv5

Built an AI tool that converts visual scenes into audio descriptions using image captioning and text-to-speech, enhancing accessibility for visually impaired.

Motivation: Automates PCB defect detection to reduce inspection time and improve accuracy by reliably identifying micro-defects in high-volume production lines.

Key Contributions

  • Converted XML annotations to YOLO format and organized a dataset of 100+ annotated PCB images covering six defect classes: Missing Hole, Mouse Bite, Open Circuit, Short, Spur, and Spurious Copper.
  • Trained a custom YOLOv5 model on the dataset with hyperparameter tuning, achieving high detection precision for all defect types.
  • Designed a severity scoring system based on defect type and bounding box size to prioritize faults visually and quantitatively.
  • Automated model evaluation and batch inference, saving outputs with labels and visualized results to disk for review.
  • Implemented a Tkinter-based GUI allowing users to upload a PCB image and visualize detected defects with bounding boxes and class specific severity scores.

Tools & Technologies

Python, PyTorch, OpenCV, YOLOv5, Tkinter, XML to YOLO conversion, PIL, NumPy, Matplotlib

Impact

Enabled fast and accurate defect detection across 100+ PCB samples with high model precision. The integrated GUI and severity scoring system support real-time deployment, minimizing human error and enhancing product quality control in manufacturing.
