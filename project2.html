Automated Highlight Generation from Soccer Matches


Built an AI-driven system to automatically generate highlight reels from full-length soccer match videos using deep learning for event detection and video summarization.


Motivation: Full soccer matches span over 90 minutes, with key moments scattered throughout. Manually reviewing and compiling highlights is time-consuming. This system identifies and ranks events like goals and fouls, generating concise highlight reels automatically. This saves time for fans, analysts, and broadcasters.

Key Contributions

  • Designed a video segmentation pipeline using the SoccerNet dataset, extracting labeled 10-second clips for key events like goals, fouls, shots, and corners.
  • Used ResNet50 to extract spatial features from video frames and GRUs to model temporal patterns for event classification.
  • Built a 3-layer GRU architecture followed by fully connected layers to classify events across five categories, improving accuracy with dataset expansion.
  • Developed a clip selection and stitching algorithm using a sliding window to include high-impact events within a 5-minute reel.
  • Collaborated on performance evaluation using confusion matrices, F1 plots, and loss curves to refine model performance.
  • Worked on stitching predicted clips into cohesive highlight videos with smooth transitions and temporal flow.

Tools & Technologies

Python, OpenCV, ResNet50, GRU, PyTorch, SoccerNet Dataset, NumPy, Matplotlib

Impact

Created an automated pipeline for match summarization that can assist broadcasters and sports analysts in generating highlight videos with less manual effort.
