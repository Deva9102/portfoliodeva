Recruiting Funnel Analytics

Analyzes an end-to-end recruitment pipeline using SQLite + SQL with a star-schema design for fact + dimension tables, generating analytics views for conversion rates, stage durations, and funnel drop-offs. CSV exports are ready to visualize in Tableau.

Motivation: Give talent teams a transparent, reproducible way to track each hiring stage, spot bottlenecks early, and make data-driven improvements.

Key Contributions

  • Designed a star-schema with candidates, applications, stage events fact & dimension tables.
  • Built SQL views for funnel counts, stage-to-stage & overall conversion, avg time-in-stage, education/location impact, and monthly cohorts.
  • Exported all views as CSV in exports/ for seamless Tableau dashboarding.
  • Documented the pipeline with an ER diagram and schema DDL for easy onboarding.
  • Delivered actionable insights on drop-offs, candidate segments, and screening delays.

Tools & Technologies

SQLite 3.45, SQL (DDL / DML / Views), Tableau (for visualization)

Sample Insights

  • Major drop-offs occur pre-interview.
  • Graduates hire at ~71 % vs ~61 % for non-graduates.
  • Semi-urban candidates convert ~77 % (urban 66 %, rural 61 %).
  • Screening is the bottleneck (~8 days avg time-in-stage).

Impact

Provided data-driven visibility into the hiring funnel enabling teams to reduce bottlenecks, improve conversion rates, and focus on the stages with the greatest ROI. CSV exports powered quick Tableau dashboards for leadership.
