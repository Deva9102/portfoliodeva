Fetal Health Risk Prediction and Patient Clustering

Developed a machine learning model to predict fetal health risks from cardiotocography data, enabling interpretable and accurate prenatal risk assessment.
Motivation: With rising concern around prenatal complications, we aimed to leverage machine learning to build a scalable and interpretable system that can proactively flag potential fetal distress and help clinicians make timely, data-driven decisions.

Key Contributions

  • Designed a custom continuous fetal risk score (0–100) using PCA to replace fixed class labels, allowing for more precise and meaningful predictions.
  • Built Ridge Regression, Random Forest, and XGBoost models from scratch in Python, and compared them with standard implementations using MAE, RMSE, and R² metrics.
  • Used SHAP explainability to uncover key clinical features driving risk, such as histogram width and variability measures.
  • Created a risk grouping pipeline using both standard and custom implementations of KMeans and Hierarchical Clustering, classifying patients into Low, Moderate, and High Risk.
  • Applied UMAP for dimensionality reduction and visualized clear cluster separations, supported by strong evaluation metrics like Silhouette Score (0.53).
  • Addressed class imbalance and redundant features through outlier filtering, stratified sampling, and correlation-based feature selection to improve model accuracy and reliability.

Tools & Technologies

Python, Scikit-learn, XGBoost, SHAP, UMAP, PCA, Ridge Regression, KMeans, Hierarchical Clustering, Matplotlib

Impact

Created a clinically interpretable and highly accurate ML pipeline that supports fetal health risk stratification and can potentially aid doctors in early diagnosis and decision-making.
